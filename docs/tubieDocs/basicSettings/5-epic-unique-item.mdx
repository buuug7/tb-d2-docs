## 毁灭 Annihilus【史诗】

属性跟毁灭基本一致属性都是, 所有技能+1-3, 增加 +1 致随机技能, 可以附魔 20 次, 只有维斯特玛场景的都瑞尔掉落, 附魔公式 `史诗毁灭 + 毁灭 + 3灵魂宝石`

- 快速施法(FCR) `+2% / 每次`
- 减少对敌人冰系抗性 `-1 / 每次`
- 减少对敌人火系抗性 `-1 / 每次`
- 减少对敌人电系抗性 `-1 / 每次`
- 减少对敌人毒系抗性 `-1 / 每次`

## 地狱火炬 Hellfire Torch【史诗】

属性跟地狱火炬基本一致属性都是, 所有技能+1-3, 增加 +1 致随机技能, 可以附魔 20 次, 只有维斯特玛场景的都瑞尔掉落, 附魔公式 `史诗地狱火炬 + 地狱火炬 + 3灵魂宝石`

- 攻击速度(IAS) `+2% / 每次`
- 最大伤害值(MAX) 以角色等级觉得 `+3 / 每次`, 满级 + 371 最大伤害
- 额外攻击准确率加成 `35% / 每次`
- 增强伤害(ED) `+20% / 每次`
